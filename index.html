Installation

Installing Fabric server

  1. Unpack fabric_server.zip into your desired destination
  2. Run "start fabric" from a command line

Installing Fabric studio

  1. Unpack fabric_studio.zip into your desired destination
  2. Run fabric.exe

Connect your databases to fabric

Run fabric studio

  1. File -> New project
  2. Click New DB Connection from the menu
  3. Choose Database type
  4. Set the server address, user and password

Define the integrated DataBase

Logical units are our magic sauce. They're micro-databases that get their data from your organization's databases

  1. Click "New Logical Unit"
  2. Name it, and make sure "auto-discovery" is checked
  3. Go through the wizard

Start querying the API

Open a web browser that has access to the machine that runs the fabric server

  1. Go to http://10.10.10.10:3213/q?table=TABLE_NAME&id=123&token=qazwsx
  2. You will get a JSON result of your table entry
